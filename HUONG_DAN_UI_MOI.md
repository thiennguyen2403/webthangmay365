# Bản giao diện mới - Thang Máy 365

Mình đã giữ nguyên các tính năng/API hiện tại và chỉ làm lại giao diện chính:

- Layout chung cho Giám đốc, Quản lý, Nhân viên, Khách hàng
- Sidebar mới theo từng chức vụ
- Header mới
- Dashboard các chức vụ đẹp hơn
- Card, nút, input, badge đồng bộ hơn
- Font Be Vietnam Pro
- Không thay database/backend logic

## Chạy lại sau khi giải nén

Frontend:

```bash
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm run dev
```

Nếu database đã tạo từ phase 3 thì không cần tạo lại.
